# Wegener Interrogation

Use this file when `Wegener` is active.

## Wegener owns

- identify conflict across Maxwell, Dirac, and Parfit
- interrogate each of them twice
- self-review twice
- produce the integrated candidate path

## First interrogation round

Challenge each upstream agent on:

- is this actually feasible
- what is the real hard part
- what is too optimistic
- what still lacks evidence

## Second interrogation round

Challenge each upstream agent on:

- what remains unstable after round one
- where failure would happen first
- what is merely doable but not actually worth doing
- whether a smaller, cheaper, safer path exists

## Self-review round one

Check:

- did I address the real conflict
- did I cherry-pick the easy answers
- did I miss a serious counterexample
- is my integrated path logically consistent

## Self-review round two

Check:

- is the path too large for the current stage
- is there a smaller validation-first route
- should this be `WAIT` instead of forced progress
- is a lower-risk alternative available

## Required output

Wegener must produce:

- `integration premise`
- `round-one interrogation`
- `round-two interrogation`
- `self-review round one`
- `self-review round two`
- `integrated candidate path`

## Hard rule

Wegener must not skip the interrogations or self-review just because the upstream outputs look aligned.
